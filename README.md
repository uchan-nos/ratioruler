# RatioRuler

データシートのグラフ読み取りに便利な比率定規です。
ウィンドウの幅に依存して目盛りの幅が変わるので、どんな大きさのグラフにも適用できます。

## 使い方

[使い方がぱっと分かる動画](https://twitter.com/uchan_nos/status/1603174127841341440) を見ると早いです。
文章で説明すると次の通りです。

1. 読み取りたいグラフが記載された文書ファイルを開いておきます。
2. 比率定規アプリケーションを起動したら、ウィンドウの左端をグラフの目盛りに合わせます。
3. ウィンドウの右端をドラッグし、「1」の軸をもう一つ右の目盛りに合わせます。
4. マウスを左右に動かし、赤線を読み取りたい場所に移動します。
5. 読み取りたい点でマウスの左ボタンをクリックすると、そのときの数値がクリップボードにコピーされます。

## スケーリング

目盛りの振り方が 1 ずつではないグラフを読み取る場合に、scale で倍率を調整できます。
scale=2 を選べば「1」の軸が値「2」に対応するようになります。

ドロップダウンの選択肢に存在しない倍率であっても、任意の値を直接記入して設定できます。

## 対数軸

log にチェックを入れると対数軸に変更できます。
対数グラフの読み取りに活用してください。
